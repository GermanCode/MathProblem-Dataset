# Math Problem Dataset
This is a math problem dataset that covers algebra and geometry topics.

In comparison to well-researched arithmetic word problem or algebra word problem,
this data set (math problems) contains geometry oriented math word problem which use algebraic
knowledge to solve (Coordinate Geometry or Analytical Geometry)
